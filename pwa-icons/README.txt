Pour une PWA fonctionnelle, veuillez créer les icônes suivantes à partir de votre logo :

1. pwa-192x192.png (192x192 pixels)
2. pwa-512x512.png (512x512 pixels) 
3. maskable-icon.png (512x512 pixels, avec une marge autour du logo pour l'affichage adaptatif)

Placez ces fichiers dans ce dossier "pwa-icons", puis copiez-les dans le dossier parent "public".

Vous pouvez utiliser des outils en ligne comme:
- https://maskable.app/editor pour créer l'icône maskable
- https://www.pwabuilder.com/imageGenerator pour générer toutes les tailles nécessaires
